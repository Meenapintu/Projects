About the Project.
Simulate the game between a CpU and a Human checker players

Instructions.
1.To run the program run code.ss in racket window.
2. the green pieces belong to cpu and red pieces can be moved by the user. To move first left click the cell
which contains the piece to be moved and then right click the destination cell which will move the piece if the move is valid.
3.Holding on the left mouse button on a cell highlights the possible final destinations for the piece in that cell.
4. Crowned cpu piece is olive in colour while that of user is violet-red in colour.To start a new game stop the 
program and then rerun the code in racket window.
